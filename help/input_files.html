Input files format

The expected input file format is CSV.

The first line should be the only line with strings to describe the data shown in each column. All the remaining lines should contain numbers with a decimal point '.' separated by comma ','.

PlotView expects 2 column of data: the first one is X data and the second one is Y data.

Below is an example of valid CSV file:
x,y
0,0
0.1,0.03
0.2,0.06
0.3,0.09

Theare can be spaces around the central comma as shown in the example test/xpower_with_spaces.csv.
